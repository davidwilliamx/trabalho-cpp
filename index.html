C++ é uma linguagem de programação de propósito geral, orientada a objetos e compilada.

História

C++ foi desenvolvido por Bjarne Stroustrup nos Bell Labs na década de 1980. O objetivo de Stroustrup era criar uma linguagem de programação que combinasse a eficiência e a flexibilidade do C com o poder da programação orientada a objetos.

Stroustrup começou a trabalhar em C++ em 1979, inicialmente chamando-a de "C com classes". As primeiras versões da linguagem eram bastante simples, mas Stroustrup continuou a desenvolvê-la e adicioná-la novos recursos ao longo dos anos.

Em 1985, a primeira versão oficial do C++ foi lançada. A linguagem foi rapidamente adotada por desenvolvedores em todo o mundo e tornou-se uma das linguagens de programação mais populares do mundo.

Eficiência

C++ é uma linguagem de programação compilada, o que significa que o código é convertido em código de máquina antes de ser executado. Isso resulta em um desempenho superior em comparação com linguagens interpretadas, como Python e JavaScript.

Flexibilidade

C++ é uma linguagem de propósito geral, o que significa que pode ser usada para criar uma ampla variedade de softwares.

Poder

C++ fornece suporte para programação orientada a objetos, que é uma abordagem de programação poderosa que permite que os desenvolvedores criem softwares mais modulares e reutilizáveis.

Atualmente, C++ é usada para criar uma ampla variedade de softwares, incluindo:

  • Sistemas operacionais: Windows, macOS, Linux, Android e iOS são todos baseados em C++.
  • Aplicativos de desktop: C++ é uma linguagem popular para desenvolvimento de aplicativos de desktop, como navegadores da web, editores de texto e software de produtividade.
  • Aplicativos móveis: C++ é uma linguagem popular para desenvolvimento de aplicativos móveis, como jogos e aplicativos de negócios.
  • Jogos: C++ é uma linguagem popular para desenvolvimento de jogos, pois oferece o desempenho e a flexibilidade necessários para criar jogos complexos e envolventes.

C++ continua a ser uma linguagem de programação importante e popular. É uma linguagem poderosa e versátil que pode ser usada para criar uma ampla variedade de softwares.

Básico

Para começar a aprender C++, é importante ter uma base sólida em programação. Se você é iniciante, recomendo que você comece aprendendo as seguintes noções básicas.

Tipos de dados

Em C++, os tipos de dados determinam o tipo de valor que uma variável pode armazenar. Os tipos de dados básicos incluem int, char, float, double e bool.

  • int: Números inteiros, como 1, 2, 3, 4, ...
  • char: Caracteres, como 'a', 'b', 'c', ...
  • float: Números de ponto flutuante, como 3.14, 2.71, ...
  • double: Números de ponto flutuante de precisão dupla, como 3.14159265358979323846, 2.71828182845904523536, ...
  • bool: Booleanos, que podem ter dois valores: true ou false.

// Declarando uma variável do tipo int

int x = 10;

// Declarando uma variável do tipo char

char y = 'a';

// Declarando uma variável do tipo float

float z = 3.14;

// Declarando uma variável do tipo double

double w = 2.71828;

// Declarando uma variável do tipo bool

bool b = true;

Operadores

Os operadores são usados para realizar operações matemáticas, lógicas e de comparação em variáveis.

  • Operadores aritméticos: +, -, *, /, %
  • Operadores lógicos: &&, ||, !, ^, ~
  • Operadores de comparação: ==, !=, <, >, <=, >=

// Usando operadores aritméticos para calcular a soma e a diferença de dois números

int x = 10;

int y = 20;

int soma = x + y; // soma = 30

int diferenca = x - y; // diferenca = -10

// Imprimindo os resultados

std::cout << "A soma de x e y é " << soma << std::endl;

std::cout << "A diferença de x e y é " << diferenca << std::endl;

// Usando operadores lógicos para verificar se um número é par ou ímpar

int x = 10;

bool par = (x % 2 == 0); // par = true

bool impar = (x % 2 != 0); // impar = false

// Imprimindo os resultados

std::cout << "O número x é par? " << par << std::endl;

std::cout << "O número x é ímpar? " << impar << std::endl;

// Usando operadores de comparação para verificar se dois números são iguais

int x = 10;

int y = 20;

bool iguais = (x == y); // iguais = false

// Imprimindo o resultado

std::cout << "Os números x e y são iguais? " << iguais << std::endl;

Entrada e saída de dados

C++ oferece várias maneiras de ler e escrever dados do usuário. Os métodos mais comuns são cin e cout.

  • cin: O operador cin é usado para ler dados do usuário. Ele pode ser usado para ler números, caracteres ou strings.
  • cout: O operador cout é usado para escrever dados na tela. Ele pode ser usado para escrever números, caracteres ou strings.

// Usando `cin` para ler um número do usuário

int x;

std::cout << "Digite um número: ";

std::cin >> x;

// Imprimindo o número digitado

std::cout << "O número que você digitou é " << x << std::endl;

// Usando `cout` para imprimir uma string

std::string nome;

std::cout << "Qual é o seu nome? ";

std::cin >> nome;

// Imprimindo o nome

std::cout << "O seu nome é " << nome << std::endl;

Estruturas de decisão

As estruturas de decisão permitem que você controle o fluxo de execução do seu programa. Elas permitem que você execute um bloco de código apenas se uma condição for verdadeira.

if: A estrutura if permite que você execute um bloco de código se uma condição for verdadeira.

// Verificando se um número é maior que 10

int x = 15;

if (x > 10) {

std::cout << "O número x é maior que 10" << std::endl;

}

if...else: A estrutura if...else permite que você execute um bloco de código se uma condição for verdadeira e outro bloco de código se a condição for falsa.

// Verificando se um número é par ou ímpar

int x = 10;

if (x % 2 == 0) {

std::cout << "O número x é par" << std::endl;

} else {

std::cout << "O número x é ímpar" << std::endl;

}

switch: A estrutura switch permite que você execute um bloco de código diferente para cada valor de uma variável.

// Verificando o dia da semana

int dia = 1;

switch (dia) {

case 1:

std::cout << "Domingo" << std::endl;

break;

case 2:

std::cout << "Segunda-feira" << std::endl;

break;

case 3:

std::cout << "Terça-feira" << std::endl;

break;

case 4:

std::cout << "Quarta-feira" << std::endl;

break;

case 5:

std::cout << "Quinta-feira" << std::endl;

break;

case 6:

std::cout << "Sexta-feira" << std::endl;

break;

case 7:

std::cout << "Sábado" << std::endl;

break;

default:

std::cout << "Dia inválido" << std::endl;

}

Estruturas de repetição

As estruturas de repetição permitem que você execute um bloco de código várias vezes.

for: A estrutura for permite que você execute um bloco de código um número específico de vezes.

// Imprimindo os números de 1 a 10

for (int i = 1; i <= 10; i++) {

std::cout << i << std::endl;

}

while: A estrutura while permite que você execute um bloco de código enquanto uma condição for verdadeira.

// Contando até 10

int i = 1;

while (i <= 10) {

std::cout << i << std::endl;

i++;

}

do...while: A estrutura do...while permite que você execute um bloco de código pelo menos uma vez, mesmo que a condição seja falsa.

// Contando até 10

int i = 1;

do {

std::cout << i << std::endl;

i++;

} while (i <= 10);

Intermediário

Após aprender as noções básicas, você pode começar a explorar conceitos mais avançados de C++. Alguns tópicos importantes incluem herança, polimorfismo entre outros.

Classes

As classes são compostas de atributos e métodos. Os atributos são variáveis que podem ser acessadas pelos objetos da classe. Os métodos são funções que podem ser usadas pelos objetos da classe.

Os atributos de uma classe podem ser públicos, protegidos ou privados. Os atributos públicos podem ser acessados por qualquer objeto. Os atributos protegidos podem ser acessados apenas por objetos da classe ou por classes derivadas. Os atributos privados só podem ser acessados por métodos da classe.

Os métodos de uma classe podem ser públicos, protegidos ou privados. Os métodos públicos podem ser chamados por qualquer objeto. Os métodos protegidos podem ser chamados apenas por objetos da classe ou por classes derivadas. Os métodos privados só podem ser chamados por métodos da classe.

Funções

Em C++, as funções são blocos de código que podem ser reutilizados em diferentes partes de um programa. Elas são uma das principais características da programação estruturada e orientada a objetos.

  • Reutilização de código: As funções permitem que o programador escreva o código uma vez e o use em vários lugares. Isso pode ajudar a reduzir a duplicata de código e a melhorar a legibilidade do código.
  • Organização do código: As funções podem ser usadas para organizar o código em blocos lógicos. Isso pode ajudar a melhorar a compreensão do código e a facilitar a manutenção.
  • Manutenibilidade do código: As funções podem ser usadas para agrupar o código relacionado. Isso pode ajudar a reduzir o esforço necessário para alterar ou corrigir o código.

Herança

Em C++, a herança é uma relação entre duas classes na qual a classe derivada herda os atributos e métodos da classe base. Isso permite que o programador reutilize código existente e crie classes mais complexas com menos esforço.

A herança pode ser usada para criar classes mais complexas com menos esforço. Por exemplo, uma classe Carro pode herdar de uma classe Veiculo. Isso significa que a classe Carro terá todos os atributos e métodos da classe Veiculo.

A herança pode ser simples ou multipla. A herança simples ocorre quando uma classe herda de apenas uma outra classe. A herança múltipla ocorre quando uma classe herda de mais de uma outra classe.

Polimorfismo

Em C++, o polimorfismo é a capacidade de usar o mesmo código para manipular objetos de diferentes tipos. Isso pode ser feito através da sobrecarga de funções ou da sobrecarga de operadores.

O polimorfismo pode ser usado para melhorar a flexibilidade de um programa. Por exemplo, uma função calcular() pode ser usada para calcular o valor de diferentes tipos de dados.

O polimorfismo pode ser ad-hoc ou paramétrico. O polimorfismo ad-hoc ocorre quando uma função pode ser usada para manipular objetos de diferentes tipos. O polimorfismo paramétrico ocorre quando uma função pode ser usada para manipular objetos de um tipo específico.

Avançado

Ao dominar os conceitos intermediários, você estará pronto para aprender tópicos mais avançados de C++.

Programação orientada a objetos

A programação orientada a objetos (POO) é uma abordagem de programação que enfatiza o uso de classes e objetos. As classes são modelos que definem os atributos e métodos de um objeto. Os objetos são instâncias de uma classe. A POO oferece uma série de vantagens em relação à programação estruturada, incluindo:

  • Reutilização de código: As classes podem ser reutilizadas em diferentes programas.
  • Organização do código: As classes podem ser usadas para organizar o código em blocos lógicos.
  • Facilidade de manutenção: As classes podem ser facilmente modificadas sem afetar o código que as usa.

Herança

A herança permite que uma classe herde os atributos e métodos de outra classe.

Polimorfismo

O polimorfismo permite que uma função ou operador seja usado com objetos de diferentes tipos.

Encapsulamento

O encapsulamento oculta os detalhes de implementação de uma classe.

Abstração

A abstração permite que você se concentre nos aspectos essenciais de um problema, ignorando os detalhes.

Manejo de memória

O gerenciamento de memória é a tarefa de atribuir e liberar memória dinamicamente. Em C++, o gerenciamento de memória é feito manualmente pelo programador.

Esse gerenciamento pode ser uma tarefa complexa e propensa a erros. Por esse motivo, muitas bibliotecas e frameworks fornecem suporte para gerenciamento de memória automático.

Alguns recursos de gerenciamento de memória automático incluem:

  • Coleções: As coleções são estruturas de dados que gerenciam a memória de seus elementos automaticamente.
  • Gestores de memória: Os gerentes de memória são responsáveis por atribuir e liberar memória dinamicamente.

Threads e processos

Os threads e processos são unidades de execução independentes que podem ser usadas para paralelizar seu código, eles são executados no mesmo espaço de memória, enquanto os processos são executados em espaços de memória separados. Outro processo importante é a paralelização pode melhorar o desempenho de seu código, mas pode ser difícil de implementar corretamente.

Thread é uma forma como um processo/tarefa de um software é divido em duas ou mais tarefas que podem ser executadas concorrentemente ("simultâneo").

POSIX threads é um padrão POSIX para threads, o qual define uma API padrão para criar e manipular threads. As bibliotecas que implementam a POSIX threads são chamadas Pthreads, sendo muito difundidas no universo Unix e outros sistemas operacionais semelhantes como Linux e Solaris.
